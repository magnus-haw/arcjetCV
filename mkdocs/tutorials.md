# Tutorials

## Colab walkthrough
Run arcjetCV in the browser using the Colab notebook:
- [Launch on Colab](https://colab.research.google.com/github/magnus-haw/arcjetCV/blob/main/tutorial.ipynb)

## Local notebook
A local copy of the tutorial lives at `tutorial.ipynb` in the repository root.
1. Ensure your environment is active (`conda activate arcjetcv` if using conda).
2. Launch Jupyter:
   ```bash
   jupyter notebook tutorial.ipynb
   ```
3. Follow the cells for installation, running the GUI, and API examples.

## Additional experiments
- Try the GUI with `tests/arcjet_test.mp4` to reproduce the processing flow.
- Capture your own calibration frames and run through the **Calibration** + **Processing** steps from the [User Manual](user-manual.md).
